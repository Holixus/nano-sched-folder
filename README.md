# nano-sched-folder
Folder operations nano-sched plugin

## data object

* opts `<Object>` options object
  * sources_folder `<String>`
  * dist_folder `<String>`


## folder.dist-clean (log, data)

Creates or cleans `<options.dist_folder>`.
